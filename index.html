What is Lyra?

Lyra is alternative PHP, that written on (prepare yourself) PHP! We develop this programming language, because we love PHP, and we want to correct many incorrect and uncomfortable things in PHP.

PHP in PHP? WHAT?

Some people said: «PHP in PHP: more slow!». If you think, that programming language interpreter in PHP is bad idea, you’re not right. Because Lyra is needn’t so much memory like different frameworks and libraries.
